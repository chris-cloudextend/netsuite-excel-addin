Available Formulas

=NS.GLATITLE(accountNumber)
Get the account name for a given account number
accountNumber - Account number or ID
Example: =NS.GLATITLE("1000")
Returns: "Cash - Operating Account"
=NS.GLABAL(subsidiary, account, fromPeriod, toPeriod, class, dept, location)
Get GL account balance for specified parameters
subsidiary - Subsidiary ID (use "" for all)
account - Account number or ID (required)
fromPeriod - Starting period name (e.g., "Jan 2025")
toPeriod - Ending period name (e.g., "Dec 2025")
class - Class ID (optional)
dept - Department ID (optional)
location - Location ID (optional)
Example: =NS.GLABAL(1, "4000", "Jan 2025", "Dec 2025", "", "", "")
Returns: 150000.00
=NS.GLABUD(subsidiary, budgetCategory, account, fromPeriod, toPeriod, class, dept, location)
Get budget amount for specified parameters
subsidiary - Subsidiary ID (use "" for all)
budgetCategory - Budget category name (e.g., "Operating")
account - Account number or ID (required)
fromPeriod - Starting period name (e.g., "Jan 2025")
toPeriod - Ending period name (e.g., "Dec 2025")
class - Class ID (optional)
dept - Department ID (optional)
location - Location ID (optional)
Example: =NS.GLABUD(1, "Operating", "6000", "Jan 2025", "Dec 2025", "", "", "")
Returns: 120000.00

Troubleshooting

  • If formulas show #NAME? error, make sure the add-in is loaded
  • If formulas return errors, check the server status above
  • Ensure the backend server is running on http://localhost:5001
  • Check your NetSuite credentials in backend/netsuite_config.json
