üìö Getting Started Guide

üí° Quick Start

Type formulas directly in any Excel cell. Start with =NS.GLABAL("4220", "Jan 2024", "Jan 2024") to get account 4220's January balance!

‚ö° Pro Tip: Load Data First

For the fastest experience, use Data Management ‚Üí Load Data before building your report:

  1. Select cells containing your date headers (e.g., Jan 2025, Feb 2025)
  2. Click Load Data for Selected Dates
  3. All GL data is pre-cached ‚Äî formulas resolve instantly!

Check "Include Balance Sheet Accounts" if you need Cash, AR, AP, etc.

Available Formulas

=NS.GLABAL(account, startPeriod, endPeriod, [subsidiary], [dept], [location], [class]) Returns: Account balance for the period
=NS.GLATITLE(account) Returns: Account name (e.g., "Revenue - Products")
=NS.GLACCTTYPE(account) Returns: Account type (e.g., "Income", "Expense")
=NS.GLAPARENT(account) Returns: Parent account number

üìã Formula Reference
NS.GLATITLE
Get the account name for any NetSuite account number or ID.
=NS.GLATITLE(accountNumber)
Parameters
accountNumber - Account number or internal ID (numeric or text)
Examples
=NS.GLATITLE(A4)
=NS.GLATITLE("4220")
=NS.GLATITLE("15000-1")
NS.GLACCTTYPE
Get the account type for any account (Income, Expense, Bank, AcctRec, Equity, etc.).
=NS.GLACCTTYPE(accountNumber)
Parameters
accountNumber - Account number or internal ID (numeric or text)
Examples
=NS.GLACCTTYPE("4220")
Returns: "Income"
=NS.GLACCTTYPE("10010")
Returns: "Bank"
NS.GLAPARENT
Get the parent account number for any sub-account (returns empty string if no parent).
=NS.GLAPARENT(accountNumber)
Parameters
accountNumber - Account number or internal ID (numeric or text)
Examples
=NS.GLAPARENT("4220")
Returns: "4210"
=NS.GLAPARENT("4000")
Returns: "" (top-level account)
NS.GLABAL
Get the GL account balance for a specific period or range, with optional filtering by subsidiary, department, location, and class.
=NS.GLABAL(account, fromPeriod, toPeriod, [subsidiary], [department], [location], [class])
Required Parameters
account - Account number or ID
fromPeriod - Starting period (e.g., "Jan 2025")
toPeriod - Ending period (e.g., "Dec 2025")
Optional Parameters
subsidiary - Filter by subsidiary ID
department - Filter by department ID
location - Filter by location ID
class - Filter by class ID
Examples
=NS.GLABAL("4010", "Jan 2025", "Dec 2025")
=NS.GLABAL(A4, B1, C1)
=NS.GLABAL("4010", "Jan 2025", "Jan 2025", "", "13", "", "")
NS.GLABUD
Get the budget amount for an account and period, with optional filtering by subsidiary, department, location, and class.
=NS.GLABUD(account, fromPeriod, toPeriod, [subsidiary], [department], [location], [class])
Required Parameters
account - Account number or ID
fromPeriod - Starting period (e.g., "Jan 2025")
toPeriod - Ending period (e.g., "Dec 2025")
Optional Parameters
subsidiary - Filter by subsidiary ID
department - Filter by department ID
location - Filter by location ID
class - Filter by class ID
Examples
=NS.GLABUD("5000", "Jan 2025", "Dec 2025")
=NS.GLABUD(A4, B1, C1)
